# Gerador Sequêncial de Fotos

Com base em uma pasta com fotos sequencias (Ex: 1.jpg, 2.gif...) gera *N* pastas com as fotos em sequencia continua.

### Exemplo:  

- Pasta 1  
  - 1.jpg  
  - 2.gif  
- Past 2  
  - 3.jpg (cópida da 1.jpg)
  - 4.gif (cópida da 2.gif)
- ...

## Dev

```bash
yarn postmerge
yarn start
```

## Release

```bash
yarn build
```
